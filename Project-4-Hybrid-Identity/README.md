# Project 4: Hybrid Identity Synchronization

## Overview
Deployed On-Premises Active Directory to Microsoft Entra ID Synchronization using Microsoft Entra Connect


## Technologies Used
- Windows Server 2022 (On-Premises Active Directory) 
- Microsoft Entra ID (Azure AD) 
- Microsoft Entra Connect Sync 
- Azure Portal 
- Active Directory Users & Computers (ADUC) 

## Implementation Steps  
1. Installed AD DS
2. Created AD Objects
3. Verified Custom Domain
4. Prepared Cloud Tenant
5. Installed Microsoft Entra Connect
6. Chose Custom Installation
7. Selected Sign-in Method
8. Connected to Entra ID
9. Configured AD Forest
10 .Configured UPN Mapping
11 .Configured OU Filtering
12 .Enabled Optional Features
13 .Completed Configuration
14 .Validated Synchronization
