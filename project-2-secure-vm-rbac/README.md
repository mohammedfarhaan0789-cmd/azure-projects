# Project 2: Secure Azure VM with RBAC & Backup

## Overview
Implemented security best practices for Azure Virtual Machines using RBAC, Network Security Groups, and backup solutions.

## Technologies Used
- Azure RBAC & IAM
- Network Security Groups (NSG)
- Azure Backup & Recovery
- Azure Monitor & Alerts
- Resource Locks

## Implementation Steps
1. Configured Role-Based Access Control (RBAC)
2. Implemented Network Security Group rules
3. Set up Azure Backup with retention policies
4. Configured monitoring and alerting
5. Applied resource locks for protection
