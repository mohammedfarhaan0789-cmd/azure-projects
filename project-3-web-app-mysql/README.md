# Project 3: Azure Web App with MySQL on Linux VM

## Overview
Deployed Azure Web App integrated with MySQL database running on Linux Virtual Machine.

## Technologies Used
- Azure Web Apps
- MySQL Database
- Linux Virtual Machine
- Virtual Networking
- App Service Plans

## Implementation Steps  
1. Created Azure Web App with App Service Plan
2. Deployed MySQL database on Linux VM
3. Configured virtual network integration
4. Set up connection string and environment variables
5. Implemented continuous deployment
